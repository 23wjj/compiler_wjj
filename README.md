# compiler_wjj
## Support
* typecast: char2int
* binary operation: +，-，*，/
* functioncall
* global/local variable declaration
* array: one dimension
* data type: int, char, double
* function: scanf, printf
* condition branch
  ```
  if{
    stmts
  }
  else{
    stmts
  }
  ```
* loop
  ```
  while(exp){
    stmts
  }
